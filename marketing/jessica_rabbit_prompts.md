## Step 3: Edit in CapCut

### Import to CapCut:
1. Import all generated images/videos
2. Add your song "bad.mp3" as audio track
3. Apply these effects:
   - **Filter**: Vintage or Film Noir
   - **Effects**: Light Leak, Sparkle, Smoke
   - **Transitions**: Fade, Zoom, Slide
   - **Text**: Add lyrics as text overlays

### CapCut Timeline:
```
Track 1: Cabaret background (full length)
Track 2: Jessica singing (sync with audio)
Track 3: Audience reactions (every 10-15 seconds)
Track 4: Text overlays (lyrics)
Track 5: Effects (hearts, sparkles, smoke)
```

### Export Settings:
- Resolution: 1080x1920 (vertical)
- Frame Rate: 30 fps
- Quality: High
